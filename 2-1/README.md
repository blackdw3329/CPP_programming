# 실습과제1
## 1. 전처리 또는 선행처리 단계를 설명하라
> 컴파일 단계 이전에 **전처리기가 소스코드를 미리 처리하는 과정**이며, 이 단계에서 선언된 **헤더파일을 소스코드에 포함**시킵니다.
> 
## 2. 객체 cin, cout은 어디에 선언되어 있는가? <br>선언된 파일을 찾아서 의미를 설명하고 캡쳐하여 첨부하라
> 객체 `cin`과 `cout`은 C++ 표준 입출력 라이브러리인 `iostream` 헤더 파일에 선언되어 있으며, `std` 네임스페이스에 포함되어 있습니다.
> `cin`은 `istream` 객체로 표준 입력을 담당하고, `cout`은 `ostream` 객체로 표준 출력을 담당합니다.

## 3. 프로그램을 링크할때 이름 충돌이 발생하는 경우를 설명해보라
> 서로 다른 파일에 같은 이름의 변수나 함수 정의되어 있으면 **링커가 어떤 것을 사용할지 구분할 수 없어 오류가 발생**합니다.

## 4. 스트림이란 무엇인가?
> 프로그램과 입출력 장치 사이의 **데이터의 이동을 위한 소프트웨어적 경로**입니다.

## 5. 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?
> C언어에서는 표준 출력 스트림을 `FILE` 자료형으로 정의하며 `stdio.h`에 선언되어 있습니다.

## 6. 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은?
> C++언어에서는 표준 출력 스트림을 `std::ostream` 클래스 로 정의하며 `iostream`에 선언되어 있습니다.

# 실습과제2 소스코드 설명
</br>

```
#include<iostream>
```
* **입출력 기능을 사용하기 위해 `iostream` 라이브러리를 포함하는 지시문입니다.**
```
using namespace std;
```
* **std 이름공간에 선언된 모든 이름에 `std::`생략합니다.**
```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수로, 프로그램의 시작 지점입니다.**

```
	cout << "이름 : 홍길동" << endl;
	cout << "주소 : 군산시 대학로 558" << endl;
	cout << "학번 : 201012" << endl;
	cout << "차종 : 벤츠" << endl;
```
* **표준 출력 객체인 `cout`을 사용하여 화면에 개인정보 문자열들을 출력하는 코드입니다.**
```
	return 0;
```
* **프로그램이 정상적으로 종료되었음을 운영체제에 알리는 코드입니다.**

# 실습과제4 소스코드 설명
</br>

```
#include<iostream>
```
* **입출력 기능을 사용하기 위해 `iostream` 라이브러리를 포함하는 지시문입니다.**
```
using namespace std;
```
* **std 이름공간에 선언된 모든 이름에 `std::`생략합니다.**
```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수로, 프로그램의 시작 지점입니다.**

```
for (int i = 1; i <= n; i++)
	{
		for (int j = 1; j <= n; j++)
		{
			if (i==j or i+j == n+1)
			{
				cout << "*";
			}
			else
			{
				cout << "  ";
			}
		}
		cout << "\n";
	}
```
* **7X7 평면을 만들어서 특정 좌표만 \*로 채우는 아이디어이며, \ 왼쪽 대각선은 왼쪽 상단이 1,1 시작이라고 할때 i,j가 같을떄 위치합니다. / 오른쪽 대각선은 7X7 인 평면에서 i+j 값이 8일때 위치합니다. 해당하지 않는 죄표는 공백으로 채워 X 모양을 출력하는 코드입니다.**
```
return 0;
```
* **프로그램이 정상적으로 종료되었음을 운영체제에 알리는 코드입니다.**
