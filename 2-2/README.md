# 실습과제1

## 1. 라이브러리는 헤더파일과 목적파일로 구성되어 있다. 헤더파일과 목적파일에 들어 있는 내용이 무엇인지 각각 설명하시오
> 헤더파일(.h)은 함수, 클래스, 변수 등의 **선언(declaration)** 이 포함된 파일로 프로그램에서 사용할 인터페이스를 제공합니다.  
> 목적파일(.obj)은 컴파일 과정에서 생성되며 **함수의 실제 정의와 실행 코드(기계어)** 가 포함되어 있습니다.

## 2. 함수의 선언, 정의, 호출은 각각 어디에 존재하는가?
> 함수의 **선언**은 보통 헤더파일(.h)에 존재한다.  
> 함수의 **정의**는 소스파일(.cpp)에 작성되며 컴파일 후 목적파일에 포함됩니다.  
> 함수의 **호출**은 main 함수와 같은 사용자 프로그램 코드에서 이루어집니다.

## 3. `<iostream>` 헤더파일에 cin, cout은 extern 변수로 선언되어 있다. extern 선언과 변수선언의 차이를 설명하라. 그리고 cin, cout의 변수선언은 어디에 있는가?
> **변수 선언(정의)** 은 메모리를 할당하여 실제 변수를 생성하는 것입니다.  
> **extern 선언**은 다른 파일에 정의된 변수를 사용하겠다는 선언으로 메모리를 생성하지 않습니다.  
> `<iostream>`에는 `cin`, `cout`이 extern으로 선언되어 있으며 실제 정의는 **C++ 표준 라이브러리 구현 파일(libstdc++ 등)** 에 존재합니다.

## 4. 라이브러리 함수나 객체를 사용하기 전에 헤더파일을 include하는 이유를 설명하라
> 헤더파일을 include 하는 이유는 **라이브러리 함수나 객체의 선언 정보를 컴파일러에게 알려주기 위해서이다.**  
> 이를 통해 컴파일러가 함수의 반환형, 매개변수, 객체 타입을 확인할 수 있습니다.  
> include 하지 않으면 컴파일 과정에서 해당 함수나 객체를 인식하지 못해 오류가 발생합니다.

# 실습과제3 소스코드 설명

```
#include <cstring>
```
* **문자열을 처리하는 함수들을 사용하기 위해 `cstring` 라이브러리를 포함하는 지시문입니다.**  
```
using namespace std;
```
* **std 이름공간에 선언된 모든 이름에 `std::`생략합니다.**
```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수로, 프로그램의 시작 지점입니다.**

```
char number[10];
char name[10];
char address[30];
```
* **학번, 이름, 주소를 저장하기 위한 문자 배열을 선언하는 코드입니다.**
```
cout << "학번을 입력하시오: ";
cin >> number;
```
* **표준 출력 객체 `cout`을 이용해 안내 문장을 출력하고, 표준 입력 객체 `cin`을 사용하여 학번을 입력받는 코드입니다.**

```
cout << "주소을 입력하시오: ";
cin.getline(address, sizeof(address), '\n');
```

```
cout << "1.학번:" << number << endl;
cout << "2.이름:" << name << endl;
cout << "3.주소:" << address << endl;
```
* **입력받은 학번, 이름, 주소 정보를 화면에 출력하는 코드입니다. `endl`은 줄바꿈을 수행합니다.**

# 실습과제4 소스코드 설명

```
using namespace std;
```
* **std 이름공간에 선언된 모든 이름에 `std::`생략합니다.**
```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수로, 프로그램의 시작 지점입니다.**

```
double num[10];
double sum = 0;
```
* **학번, 이름, 주소를 저장하기 위한 문자 배열을 선언하는 코드입니다.**
```
cout << "학번을 입력하시오: ";
cin >> number;
```
* **입력받은 실수를 저장하기 위한 배열 `num`을 선언하고, 양수의 합을 저장할 변수 `sum`을 0으로 초기화합니다.**
```
cout << "실수 5개를 입력하세요>> ";
```
* **실수 5개를 입력하도록 안내하는 문장을 출력하는 코드입니다.**
```
for (int i = 0; i < 5; i++)
{
  cin >> num[i];
}
```
* **반복문을 사용하여 실수 5개를 입력받아 배열 `num`에 저장하는 코드입니다.**

```
for (int j = 0; j < 5; j++)
{
  if (num[j] > 0)
  {
    sum += num[j];
  }
}
```
* **배열에 저장된 값들을 검사하여 양수일 경우 `sum` 변수에 누적하여 더하는 코드입니다.**
```
cout << "양수의 합: " << sum << endl;
```
* **계산된 양수들의 합을 화면에 출력하는 코드입니다. `endl`은 줄바꿈을 수행합니다.**
```

```

```
#include <cstring>
```
* **문자열을 처리하는 함수들을 사용하기 위해 `cstring` 라이브러리를 포함하는 지시문입니다.**
```
using namespace std;
```
* **std 이름공간에 선언된 모든 이름에 `std::`생략합니다.**
```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수로, 프로그램의 시작 지점입니다.**

```
char text[100];
```
* **입력한 문장을 저장하기 위한 문자 배열 `text`를 선언하는 코드입니다.**
```
while(true)
```
* **특정 조건을 입력할 때까지 프로그램을 반복 실행하기 위한 무한 반복문입니다.**
```
cin.getline(text, sizeof(text));
```
* **입력한 한 줄의 문자열을 `text` 배열에 저장하는 코드입니다. 공백을 포함한 문자열을 입력받을 수 있습니다.**
```
if(strcmp(text, "exit") == 0) break;
```
* **입력된 문자열이 `"exit"`과 같은지 비교하여 같으면 반복문을 종료하고 프로그램을 끝내는 코드입니다.**

```
int count = 0;
```
* **문장 속 단어의 개수를 저장하기 위한 변수 `count`를 0으로 초기화하는 코드입니다.**
```
for(int i = 0; text[i] != '\0'; i++)
```
* **문자열의 끝을 나타내는 널 문자(`\0`)가 나올 때까지 문자열의 각 문자를 검사하기 위한 반복문입니다.**
```
if(text[i] != ' ' && (i == 0 || text[i-1] == ' '))
{
count++;
}
```
* **현재 문자가 공백이 아니고, 문자열의 시작이거나 앞 문자가 공백이면 새로운 단어의 시작으로 판단하여 단어 개수를 증가시키는 코드입니다.**
```
cout << "단어 개수: " << count << endl;
```
* **입력된 문장에 포함된 단어의 개수를 화면에 출력하는 코드입니다. `endl`은 줄바꿈을 수행합니다.**
```
return 0;
```
