# 실습과제2

## 1. 매크로상수 `NULL`이 무엇인지 조사하라
> `NULL`은 포인터가 아무 대상도 가리키지 않음을 나타내는 상수입니다. C에서는 `0` 또는 `(void*)0`으로 정의되고, C++에서는 주로 `0`으로 정의됩니다. `<stddef.h>`, `<stdio.h>`, `<stdlib.h>`, `<cstddef>` 등에 정의되어 있으며 널 포인터 표현에 사용됩니다.

## 2. 널문자와 `NULL`의 차이를 설명하시오
> 널문자 `'\0'`는 문자열의 끝을 나타내는 문자이고, `NULL`은 비어 있는 포인터를 의미합니다. 널문자는 `char`형(1바이트)이며 값은 0이고, `NULL`은 포인터 개념으로 주소가 없음을 나타낼 때 사용됩니다.

## 3. `nullptr`이 무엇인지 조사하시오
> `nullptr`은 C++11에서 도입된 널 포인터 전용 키워드입니다. `NULL`은 정수 0으로 처리될 수 있어 혼동이 있지만, `nullptr`은 포인터 타입으로 명확하게 구분되어 더 안전하게 사용됩니다.

# 실습과제3 소스코드 설명
</br>

```
#include <iostream>
```
* **입출력 기능을 사용하기 위해 `iostream` 라이브러리를 포함합니다.**

```
using namespace std;
```
* **`std::`를 생략하고 표준 라이브러리를 사용하기 위해 선언합니다.**

```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수입니다.**

```
int n;
int total = 0;
double avg;
```
* **입력 개수(`n`), 합계(`total`), 평균(`avg`)을 저장할 변수를 선언합니다.**

```
cout << "입력할 정수의 개수를 입력하시오:";
cin >> n;
```
* **사용자로부터 입력할 정수의 개수를 입력받습니다.**

```
int* p = new int[n];
```
* **입력받은 개수만큼 동적 메모리를 할당하여 배열을 생성합니다.**

```
cout << n << "개의 정수를 입력 하시오." << endl;
```
* **입력 안내 메시지를 출력합니다.**

```
for (int i = 0; i < n; i++)
{
	cin >> p[i];
}
```
* **동적 배열에 정수 값을 입력받습니다.**

```
for (int j = 0; j < n; j++)
{
	total += p[j];
}
```
* **배열의 모든 값을 더하여 총합을 계산합니다.**

```
avg = total / n;
```
* **총합을 개수로 나누어 평균을 계산합니다.**

```
cout << "평균값은 " << avg << "입니다" << endl;
```
* **계산된 평균 값을 출력합니다.**

```
delete[] p;
```
* **동적으로 할당한 메모리를 해제합니다.**

# 실습과제4 소스코드 설명

```
#include <iostream>

```
```
using namespace std;
```
* **`std::`를 생략하고 표준 라이브러리를 사용하기 위해 선언합니다.**

```
int main()
```
* **프로그램의 실행이 시작되는 메인 함수입니다.**

```
int n;
int max;
```
* **입력 개수(`n`)와 최대값(`max`)을 저장할 변수를 선언합니다.**

```
cout << "입력할 정수의 개수를 입력하시오:";
cin >> n;
```
* **사용자로부터 입력할 정수의 개수를 입력받습니다.**

```
int* p = new int[n];
```
* **입력 개수만큼 동적 배열을 생성합니다.**

```
cout << n << "개의 정수를 입력 하시오." << endl;
```
* **입력 안내 메시지를 출력합니다.**

```
for (int i = 0; i < n; i++)
{
	cin >> p[i];
}
```
* **배열에 정수 값을 입력받습니다.**

```
max = p[0];
```
* **첫 번째 값을 초기 최대값으로 설정합니다.**

```
for (int j = 0; j < n; j++)
{
	if (max <= p[j]) max = p[j];
}
```
* **배열을 순회하며 더 큰 값을 찾으면 최대값을 갱신합니다.**

```
cout << "최대값은 " << max << "입니다" << endl;
```

```

```
* **동적 할당된 메모리를 해제합니다.**
